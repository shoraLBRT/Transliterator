# Transliterator

**Transliterator** — это инструмент для транслитерации арабского текста с огласовками в расширенную русскую кириллицу. Проект разработан с учетом возможности будущего расширения: поддержка различных правил таджвида, кастомных профилей транслитерации и потенциально других целевых языков.

---

## Особенности

* Транслитерация арабского текста с огласовками в русскую расширенную кириллицу.
* Обработка некоторых правил таджвида, включая васлю, алиф максура и эмфатические буквы.
* Возможность создания и использования **кастомных профилей транслитерации**: менять соответствие букв под конкретные нужды.
* Разработан как **CLI-приложение**, с перспективой добавления API в будущем.
* Юнит-тесты для отдельных правил обеспечивают контроль корректности транслитерации.

> На данный момент реализованы основные правила, функциональность будет расширяться по мере добавления новых правил таджвида и профилей.

---

## Пример работы

**Входной текст (арабский с огласовками):**

```
بِسْمِ ٱللَّهِ ٱلرَّحْمَـٰنِ ٱلرَّحِيمِ ١ ٱلْحَمْدُ لِلَّهِ رَبِّ ٱلْعَـٰلَمِينَ ٢ ٱلرَّحْمَـٰنِ ٱلرَّحِيمِ ٣ مَـٰلِكِ يَوْمِ ٱلدِّينِ
```

**Результат транслитерации:**

```
бисми-лляhи-ррохIмаани-ррохIийм 1 аль-хIамду лилляhи робби-ль'аалямиин 2 ар-рохIмаани-ррохIиим 3 маалики йауми-ддиин
```

---

## Использование

На данный момент проект реализован как **CLI-приложение**, принимающее строку арабского текста и возвращающее транслитерированный текст.

Пример использования:

```bash
dotnet run "بِسْمِ ٱللَّهِ ٱلرَّحْمَـٰنِ ٱلرَّحِيمِ"
```

Вывод:

```
бисми-лляhи-ррохIмаани-ррохIийм
```

> В будущем будет добавлена возможность выбора профиля транслитерации, работы с файлами и API.

## Кастомные профили

* Профили позволяют определять, как каждая арабская буква и огласовка транслитерируются.
* Профили могут быть созданы для индивидуальных нужд пользователя или для других целевых языков.
* Использование профилей позволяет расширять функциональность без изменения кода ядра транслитератора.

Пример профиля:

        {"ى", "а"},
        {"ٰ", "а"},
        {"ۤ", ""},
        {"ٲ", "а"},
        {"ٳ", "а"},
        {"ب", "б"},
        {"ت", "т"},
        {"ث", "с́"},
        {"ج", "дж"},
        {"ح", "хI"},
        {"خ", "хъ"},
        {"د", "д"},
        {"ذ", "зъ"},
        {"ر", "р"},
        {"ز", "з"},
        {"س", "с"},
        {"ش", "щ"},
        {"ص", "сI"},
        {"ض", "дI"},
        {"ط", "тI"},
        {"ظ", "зI"},
        {"ع", "'"},
        {"غ", "гъ"},
        {"ف", "ф"},
        {"ق", "q"},
        {"ك", "к"},
        {"ل", "л"},
        {"م", "м"},
        {"ن", "н"},

## Юнит-тесты

* В проекте частично реализованы **юнит-тесты для отдельных правил транслитерации**, чтобы проверять корректность обработки васли, алиф максуры и других особенностей.
* Тесты написаны с использованием **xUnit**.
* Для запуска тестов в Visual Studio:

  1. Открой `Test Explorer` (`Test → Test Explorer`).
  2. Собери решение (`Build → Build Solution`).
  3. Нажми `Run All` или выбери отдельный тест для запуска.

> Тесты помогают убедиться, что изменения правил транслитерации не ломают существующую функциональность.
